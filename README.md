# 🧬 BiomedDeadlines

Welcome to **BiomedDeadlines**, a clean, automated, and crowdsourced tracker for biomedical, clinical, and biotech conference abstract deadlines.

You can visit the live website here: [(https://jambuzz.github.io/biomeddeadlines/)]

## 🌟 Features
* **Live Search:** Instantly search by conference name, city, or keyword.
* **Smart Filters:** Sort by subfield (Oncology, Neuro, Cardiology, etc.) and target audience (Undergrads, Med Students, Clinicians, etc.).
* **Automated Monitoring:** Powered by a daily Python script that monitors official medical society websites for updates.

## 🤝 How to Contribute a Conference
Missing a major conference? You can easily add it!
1. Open the `conferences.json` file in this repository.
2. Click the pencil icon to edit.
3. Copy an existing conference block and paste it at the bottom, filling in your new conference's details.
4. Click "Commit changes." The website will update instantly!

*(Alternatively, you can email suggestions to: biomeddeadlines@gmail.com)*
